# SPEED TEST
Бесплатное веб-приложение для анализа скорости интернет-соединения в браузере

![](./application-%20illustration.png)

### Авторы: 
* Луконин Егор Дмитриевич
* Михайлов Валерий Алексеевич


## Функционал
| Название | Описание |
| ---------| ----------- |
| Upload   | Измерение скорости выгрузки файлов|
| Download | Измерение скорости загрузки файлов|
| Ping     | Измерение времени задержки ответа от сервера|


## Локальное развертывание

#### Перед развертыванием убедитесь, что на вашем устройстве установлен Python 3.14 или выше, PIP 25.3 или выше. Если вы заметили, что он данное приложение можно запустить на версии ниже, то просим сообщить об этом в [поддержку](https://t.me/GU_B_ER)
1. Клонирование репозитория <br>
``` bash
git clone https://github.com/EgorLukonin/speed-test.git 
```
2. Переход в директорию speed-test <br>
``` bash
 cd speed test 
```
3. Установка зависимостей <br>
``` bash
pip install requirements.txt
```
4. Запуск сервера
``` bash
python3 main.py
```
